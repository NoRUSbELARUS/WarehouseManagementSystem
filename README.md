# Warehouse Management System

**Warehouse Management System** — это приложение для учета товаров, складов, поставок и заказов, предоставляющее интерфейс для работы с товарами, складами, категориями, Поставщиками, заказами, сотрудниками, ячейками хранения.

#### 1.2. Расчет оценки времени

**Формула:**  
`E = (P + O + 4 × BG) / 6`

где:
- **P** — пессимистичная оценка
- **O** — оптимистичная оценка
- **BG** — наиболее вероятная оценка

**Исходные данные:**

| Параметр | Значение (часы) |
|----------|-----------------|
| Оптимистичная оценка (O) | 2 |
| Наиболее вероятная (BG) | 4 |
| Пессимистичная оценка (P) | 8 |

**Расчет:**
`E = (9 + 2 + 4 × 4) / 6 = 4.5`

**Результат:** ожидаемое время выполнения этапа — **4.5 часа**.
**Фактически затраченно время:** — **3 часа**.
